================================================================================
                    UNIAIVERSE WEBSITE - SETUP INSTRUCTIONS
================================================================================

PROJECT: Next.js Website for uniAiverse AI Consulting
TECH STACK: Next.js 16 + React 19 + TypeScript + Tailwind CSS
COLOR THEME: Dark Blue (#0F172A), Primary Blue (#1E40AF), Light Blue (#3B82F6), White

================================================================================
                          QUICK START (3 STEPS)
================================================================================

STEP 1: Open Terminal/Command Prompt
   Navigate to the project folder:
   
   cd uniaiverse-website

STEP 2: Install Dependencies
   
   npm install
   
   This will install all required packages (Next.js, React, TypeScript, etc.)
   Wait 1-2 minutes for installation to complete.

STEP 3: Start Development Server
   
   npm run dev
   
   Open your browser and go to: http://localhost:3000

================================================================================
                          CUSTOMIZATION GUIDE
================================================================================

CHANGE COLORS:
  Edit: tailwind.config.js
  Find the "colors" section and modify hex values

EDIT CONTENT:
  Navigate to: components/ folder
  Open any .tsx file and edit text, images, or descriptions

ADD IMAGES:
  1. Place images in: public/ folder
  2. Use in components: <Image src="/your-image.jpg" />

CHANGE FONTS:
  Edit: app/globals.css
  Modify the font-family in the body section

================================================================================
                          TROUBLESHOOTING
================================================================================

ISSUE: Port 3000 already in use
SOLUTION:
  Windows: 
    netstat -ano | findstr :3000
    taskkill /PID <PID> /F
  
  Mac/Linux:
    lsof -ti:3000 | xargs kill -9

ISSUE: Module not found errors
SOLUTION:
  Delete node_modules and reinstall:
    rm -rf node_modules package-lock.json
    npm install

ISSUE: TypeScript errors
SOLUTION:
  Run: npm run lint
  Fix reported errors in component files

ISSUE: Styling not working
SOLUTION:
  1. Check if Tailwind is properly configured
  2. Restart development server: Ctrl+C then npm run dev
